### 1.[最长回文子串](https://leetcode-cn.com/problems/longest-palindromic-substring/submissions/)

输入: "babad"
输出: "bab"，注意: "aba" 也是一个有效答案。
思路：中心扩展法。
```java
class Solution {
    public String longestPalindrome(String s) {
        if(s==null || s.length()<1){
            return "";
        }
        //中心扩展法
        int len = s.length();
        int start = 0;
        int end = 0;
        for(int i=0;i<len;i++){
           int p = size(s,i,i);
           int p1 = size(s,i,i+1);
            //这道题里面，没有必要做p和p1的区分，因为都是对下标是i的操作
           int max = Math.max(p,p1);
           if(max>end-start){
               start = i-(max-1)/2;
               end = i+max/2;
           }
            
        }
        return s.substring(start,end+1);
    }
    private int size(String s,int L,int R){
        int len = s.length();
       while( L>=0 && R <len && s.charAt(L) == s.charAt(R)){
           --L;
           ++R;
       }
        //为什么-1，不为什么，问就是试出来的
        return R-L-1;
        
    }
}
```

### 2.[整数反转](https://leetcode-cn.com/problems/reverse-integer/)
输入: 123
输出: 321
```java
    public int reverse(int x) {
        long res = 0;
        while(x!=0){
            int t = x%10;
            res = res*10+t;
            x/=10;
        }
        if(res<Math.pow(-2,31) || res>Math.pow(2,31)-1){
            return 0;
        }
        return (int)res;  
    }
```
